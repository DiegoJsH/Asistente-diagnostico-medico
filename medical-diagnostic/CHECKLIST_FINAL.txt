🚀 CÓMO USAR LOCALMENTE
================================================================================

1. CLONAR DESDE GITHUB:
   $ git clone https://github.com/tu-usuario/medical-diagnostic.git
   $ cd medical-diagnostic

2. INSTALAR DEPENDENCIAS (primera vez):
   $ cd python
   $ pip install -r requirements.txt
   $ cd ../scala
   $ sbt compile

3. EJECUTAR (dos terminales):

   TERMINAL 1:
   $ cd python
   $ python3 app.py
   ✓ Escuchando en http://localhost:5000

   TERMINAL 2:
   $ cd scala
   $ sbt run
   ✓ Activo en http://localhost:8080

4. ACCEDER:
   $ Abre navegador en http://localhost:8080
