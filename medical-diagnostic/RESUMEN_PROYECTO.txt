REQUISITOS PREVIOS:
  • Scala 2.13+ con SBT
  • Python 3.8+
  • SWI-Prolog

INSTALACIÓN (una sola vez):
  $ cd python && pip install -r requirements.txt
  $ cd ../scala && sbt compile

EJECUCIÓN (en dos terminales):

  Terminal 1:
  $ cd python
  $ python3 app.py
  > Iniciando servidor Python...
  > Escuchando en http://localhost:5000

  Terminal 2:
  $ cd scala
  $ sbt run
  > [OK] Servidor Scala activo en http://localhost:8080
  > Abre: http://localhost:8080

ACCEDER:
  Abre navegador: http://localhost:8080

🔗 ENDPOINTS API DISPONIBLES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

SCALA (http://localhost:8080)
  GET  /                    → Página HTML
  GET  /api/status          → Estado del sistema
  GET  /api/symptoms        → Lista de síntomas
  GET  /api/diseases        → Lista de enfermedades
  POST /api/diagnose        → Diagnóstico (JSON)

PYTHON (http://localhost:5000)
  GET  /health              → Verificar disponibilidad
  GET  /symptoms            → Síntomas (directo Prolog)
  GET  /diseases            → Enfermedades (directo Prolog)
  POST /diagnose            → Diagnóstico (directo Prolog)

EJEMPLO curl:
  curl -X POST http://localhost:8080/api/diagnose \
    -H "Content-Type: application/json" \
    -d '{"symptoms": ["fiebre", "tos", "fatiga", "cuerpo_adolorido"]}'

ENFERMEDADES Y REGLAS (7):
  1. Gripe: fiebre + tos + fatiga + cuerpo_adolorido (95%)
  2. Resfriado: congestión_nasal + tos + estornudos (85%)
  3. Faringitis: dolor_garganta + fiebre + dolor_cabeza (80%)
  4. Gastroenteritis: vómitos + diarrea + fatiga (90%)
  5. Varicela: erupciones_piel + fiebre + fatiga (85%)
  6. Alergia: estornudos + congestión_nasal + erupciones_piel (75%)
  7. Migraña: dolor_cabeza + fatiga (70%)
