CÓMO PROCEDER:
==============

1. TERMINA el servidor Python actual (Ctrl+C)

2. LIMPIA dependencias antiguas (OPCIONAL pero recomendado):
   pip uninstall pyswip -y
   pip uninstall Flask -y

3. REINSTALA las dependencias:
   cd python
   pip install -r requirements.txt

4. INICIA Python nuevamente:
   python3 app.py

Deberías ver:
   ✓ Starting Python server...
   ✓ Listening on http://localhost:5000
   ✓ SIN ERRORES de Prolog

5. En otra terminal, inicia Scala:
   cd scala
   sbt run

Deberías ver:
   ✓ [OK] Server Scala active on http://localhost:8080

6. Abre navegador:
   http://localhost:8080

¡LISTO!

Si aún hay errores, avísame qué dice exactamente.
